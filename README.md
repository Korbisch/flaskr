# flaskr
This is a small Python Flask Blog practice project

## How do I run this?

1. Clone or fork the project    
2. Then run:    
`export FLASK_APP=flaskr`      
`export FLASK_ENV=development`     
`flask run`    

3. Visit the page at:    
http://127.0.0.1:5000/
